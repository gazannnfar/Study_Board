# Курсовые документы (StudyBoard)

Формальный комплект проектных документов по ГОСТ, актуализированный под текущую реализацию (релиз **v0.1.0**).

В каждом документе сохранён исходный текст (плановый замысел) и добавлен раздел
**«Актуализация под текущую реализацию (релиз v0.1.0)»** с фактическим состоянием:
6 ролей RBAC, реальные сущности и REST API (`/api`), расписание, локальное планирование,
PERT, AI-помощник (AI_STRICT/AI_LIGHT + реальный LLM через OpenAI-совместимый API), KPI-аналитика.
Титульные листы не изменялись.

| Файл | Документ |
| --- | --- |
| `Устав_проекта_StudyBoard.docx` | Устав проекта |
| `Техническое_задание_StudyBoard.docx` | Техническое задание (ГОСТ 34.602-2020) |
| `План_управления_и_RACI_StudyBoard.docx` | План управления проектом + матрица RACI |
| `Реестр_рисков_User_Stories_Backlog_StudyBoard.docx` | Реестр рисков, User Stories, Use Cases, Product Backlog |
| `Отчёт_Архитектор_Backend_API_StudyBoard.docx` | Отчёт по ролям: Архитектор ПО, Backend, API-developer |
| `Отчёт_PO_PM_BA_StudyBoard.docx` | Отчёт по ролям: Владелец продукта, Менеджер продукта, Бизнес-аналитик |
| `seq_studyboard.puml` | Диаграмма последовательности (исходник PlantUML, эндпоинты `/api`) |
