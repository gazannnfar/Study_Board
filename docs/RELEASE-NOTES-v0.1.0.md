# Release Notes v0.1.0

Initial educational MVP:

- React SPA Kanban board.
- Express REST API with JWT and RBAC.
- PostgreSQL schema with Prisma.
- Demo groups, users and tasks.
- Schedule import and local planning.
- PERT estimates.
- AI assistant modes: `AI_STRICT` and `AI_LIGHT`.
- KPI analytics, notifications, comments and grading.
