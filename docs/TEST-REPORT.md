## AI Assistant

Two implementations behind one `AiAssistantService` interface, switched via `AI_PROVIDER`:

- `rule` (default) — deterministic rule-based, offline, demo-safe.
- `llm` — real LLM through an OpenAI-compatible API (Groq/OpenRouter/OpenAI/Ollama).
  Prose is generated by the model; PERT numbers stay deterministic. Falls back to
  rule-based automatically when the key is missing or a request fails (verified:
  bad key -> HTTP 401 -> graceful fallback to a valid suggestion).
