# Git Flow and Conventional Commits

Branches:

- `main`: stable demo/release branch.
- `develop`: integration branch.
- `feature/<name>`: feature work.
- `fix/<name>`: bug fixes.

Commit examples:

- `feat(api): add schedule import endpoint`
- `feat(web): show planning recommendations`
- `fix(rbac): restrict schedule mutation`
- `test(ai): cover strict and light PERT modes`
- `docs: add deployment guide`
