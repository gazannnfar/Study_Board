# User Stories

1. As a student, I want to see my assigned tasks and recommended work slots so I know what to do first.
2. As a starosta, I want to assign tasks and import schedule so group work matches real study load.
3. As a teacher, I want to review progress, comments and grades so I can evaluate the team.
4. As a project manager, I want to see task status and overdue risks across groups.
5. As a team lead, I want to plan tasks into free slots and reduce deadline conflicts.
6. As an admin, I want to manage users and roles.
